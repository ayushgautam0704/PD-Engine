### PD-Engine
C++ ODIENT based Differential Equation Analyzer

### Description
- Delivered a high-performance PDE analyzer with a 30% reduction in computation time, leveraging expertise in scalable solution development and advanced optimization techniques
  and leveraging advanced libraries like ODIENT and Medusa in C++.
- Engineered a high-performance solution that solved complex PDEs 1.9x faster than traditional methods by utilizing parallel processing and efficient memory management.

### Technologies
- C++ Boost Libraries
- ODIENT & MEDUSA
- Python Libraries (Matplot & Seaborn): Data Visualization
